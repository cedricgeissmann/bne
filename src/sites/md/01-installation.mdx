## Installation

<section>

In der Informatik arbeiten wir direkt auf dem eigenen Gerät auf lokalen Dateien.
Dabei Arbeiten wir in kleinen Projekten, die jeweils in einem Ordner enthalten
sind. Für diese Arbeitsweise brauchen wir 3 Programme:

1. <a href="https://code.visualstudio.com/Download">VS Code</a> (Texteditor)
2. <a href="https://git-scm.com/downloads">Git</a> (Versionierungs-Werkzeug)
3. <a href="https://www.google.com/chrome/">Chrome</a> (Webbrowser)

</section>

<Example title="Auftrag">

Installieren Sie die Programme über die jeweiligen Links.

</Example>

### VS Code Plugins

<section>

Wenn Sie VS Code heruntergeladen haben, müssen Sie noch einige Plugins
installieren. Öffnen Sie VS Code und drücken Sie <pre>CTRL + P</pre> und geben
den folgenden Code, jeweils Zeile für Zeile ein:

```text
ext install ritwickdey.LiveServer
ext install mhutchie.git-graph
ext install esbenp.prettier-vscode
```

</section>

### Neues Projekt

Ein Projekt sollte immer unter Versionskontrolle sein. Oftmals ist es auch
möglich von einem bestehenden Projekt zu starten, und dann neue Versionen
hinzuzufügen. Das geht viel einfacher als es klingt. Sie müssen nur ein neues
Projekt mit <pre>Git</pre> klonen. Drücken Sie dazu <pre>SHIFT + CTRL + P</pre>
und geben Sie <pre>klonen</pre> ein.

Klonen Sie das folgende Projekt
(<a href="https://github.com/gymmu/01-intro-websites">https://github.com/gymmu/01-intro-websites</a>)
in einen neuen Ordner, am besten auf Ihrem Desktop.

Folgen Sie dabei am besten dem Video. Bei Unklarheiten bitte direkt fragen.

<Video url="pZ9NPGK90YA" />

Falls Sie in <pre>Git</pre> keinen commit machen können, folgen Sie der
Anleitung in diesem Video.
