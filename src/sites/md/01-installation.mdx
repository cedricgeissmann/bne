## Installation

<section>

In der Informatik braucht es viele verschiedenen Werkzeuge um mit dem Computer
und auch mit Code zu arbeiten. Diese Werkzeuge müssen zuerst installiert werden.
Im folgenden Abschnitt finden Sie eine Installationsanleitung für Ihr
Betriebssystem. Stellen Sie sicher dass Sie das richtige Betriebssystem
ausgewählt haben, ansonsten können Sie die Installation garnicht durchführen.

    Am einfachsten geht die Instalation auf Windows mit dem Kommandozeilen-Befehl `winget`. Dafür müssen Sie ein neues Terminal öffnen. Das machen Sie am besten als Administrator, damit die Installation auch sicher funktioniert.

    öffnen Sie das Startmenu und suchen Sie nach `cmd`. Mit einem Rechtsklick auf das Programm, können Sie dann `Als Administrator öffnen` auswählen. Dann sollte ein neues Fenster aufgehen, wo Sie Befehle ausführen können.

    ### VSCode

    Als erstes installieren wir VSCode. Das ist unser Editor den wir zum schreiben und editieren von Code verwenden werden. Um VSCode zu installieren, geben Sie den folgenden Befehl ein, und drücken Sie anschliessend auf `Enter`. Stellen Sie sicher das der Befehl richtig geschrieben ist, ansonsten wird es nicht funktionieren.

    ```cmd
    winget install -e --id Microsoft.VisualStudioCode
    ```

### VS Code Plugins

<section>

Wenn Sie VS Code heruntergeladen haben, müssen Sie noch einige Plugins
installieren. Öffnen Sie VS Code und drücken Sie `CTRL + P` und geben den
folgenden Code, jeweils Zeile für Zeile ein:

```text
ext install ritwickdey.LiveServer
ext install mhutchie.git-graph
ext install esbenp.prettier-vscode
```

</section>

### Neues Projekt

Ein Projekt sollte immer unter Versionskontrolle sein. Oftmals ist es auch
möglich von einem bestehenden Projekt zu starten, und dann neue Versionen
hinzuzufügen. Das geht viel einfacher als es klingt. Sie müssen nur ein neues
Projekt mit `Git` klonen. Drücken Sie dazu `SHIFT + CTRL + P` und geben Sie
`klonen` ein.

Klonen Sie das folgende Projekt
(<a href="https://github.com/gymmu/01-intro-websites">https://github.com/gymmu/01-intro-websites</a>)
in einen neuen Ordner, am besten auf Ihrem Desktop.
